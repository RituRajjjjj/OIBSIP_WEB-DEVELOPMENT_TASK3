# OIBSIP_WEB-DEVELOPMENT_TASK3
This is a simple Temperature Converter project made using HTML, CSS and JavaScript.

It can convert:
- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Celsius to Kelvin
- Kelvin to Celsius
- Fahrenheit to Kelvin
- Kelvin to Fahrenheit

How to use
1. Enter the temperature.
2. Select the conversion type.
3. Click on Convert button.
4. The converted temperature will be displayed on the screen.

Files used:
- index.html
- style.css
- script.js
